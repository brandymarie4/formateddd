## 安装设置samba
> * sudo apt-get install samba
> * sudo useradd xxx
> * sudo smbpasswd -a xxx
> * sudo vi /etc/samba/smb.conf

#### 配置内加入以下内容
```sh
[dev]
comment = dev
path = /var/dev
valid user = xxx
write list = xxx
create mask = 0664
directory mask = 0775
force user = xxx
force group = xxx
public = no
available = yes
browseable = yes
security = user
```
### 重启samba【sudo service smbd restart】


## Ubuntu分区

> * /boot	200M左右	ext4

### swap

```sh
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile

sudo vim /etc/fstab
/swapfile none swap sw 0 0
```
